# Minimal Config
## I decided to try a simpler config...
Will it be good? Probably yes and will also be very faster! 
I noticed that my Neovim loading time on my phone is really slow 
because i have a lot of plugins installed and probably running 
all at once. As my phone is a little limited in hardware, that is 
being so much for it to handle...

# Package Manager
## Lazy (dot) NVIM
I use this one because i'm not on Neovim v0.12... This is a `dev` 
version. To use it i need to compile the source code manually, and
yeah, it takes very long to finish... But we have good news! On 
v0.12 we will have a native (and probably, better, faster and 
easier to use) package/plugin manager in Neovim!

# Plugins
## As less as possible, for the best performance!
We have to agree that there are some essencial plugins. Like 
Tree Sitter, Telescope, CMP, Auto Pairs... And maybe Mason or 
None LS or Zero LSP... But we can't forget that Neovim has native 
support for LSP now (in v0.10+ as i remember)
